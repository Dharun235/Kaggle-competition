# ELIVA25-IndustrialVision

## Competition Overview
Description
In this challenge, you need to implement an industrial vision system. The goal is to detect the teeth of mechanic gear. The following figure shows a sample image with the ground-truth boxes (in blue) that have been manually annotated.
<img width="463" height="418" alt="image" src="https://github.com/user-attachments/assets/ff158198-82f9-40fe-a2eb-bbb182e752cb" />
The link to project is here - https://www.kaggle.com/competitions/eliva-25-industrial-vision/overview

## Project Structure
- `data/` - Dataset files
- `notebooks/` - Jupyter notebooks for analysis and modeling
- `output/` - Model outputs and submission files
- `visuals/` - Visualizations and plots
